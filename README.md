# A---Frog-1
https://atcoder.jp/contests/dp/tasks/dp_a

## 概要
動的計画法を使用する問題。
ジャンプしたところの差の合計をどれだけ小さくできるか。

## 方針
１マスジャンプする時のa : ２がアクセル不可能の時
２マスジャンプする時のb : b < aの時

## メモ
一部でどっちを選んだ方が良いか選ぶのではなく、全体で見たときに一つ先を選ぶのが有利か、二つ先を選ぶ方が有利か考えてとく。
abs で絶対値求めている。
INT_MAXとは無限と設定するために必要なもの。最大の数が何になるかわからないので使用する。
アクセスできないものをアクセスしようとしない。
